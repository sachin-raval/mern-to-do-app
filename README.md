# 📝 MERN To-Do App

A full-featured to-do list application built with the MERN stack.

## 🚀 Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- Tailwind CSS

## ✨ Features
- ✅ User authentication
- ✅ Add, update, delete tasks
- ✅ Mark tasks as complete
- ✅ Responsive UI

```bash
# Backend
cd server
npm install

# Frontend
